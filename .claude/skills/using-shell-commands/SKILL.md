# Using Shell Commands

Before using standard GNU/Linux commands, check if a modern Rust alternative is installed:

| Instead of | Use |
|------------|-----|
| `find` | `fd` |
| `grep` | `rg` (ripgrep) |
| `du` | `diskus` |
